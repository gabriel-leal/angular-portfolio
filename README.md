# Portfólio - Gabriel Leal Menezes

Projeto de portfólio profissional em **Next.js + TypeScript + Tailwind CSS**, com tema escuro/claro, animações leves, filtros de projetos e formulário de contato.

## Rodando localmente

```bash
npm install
npm run dev
```

Abra [http://localhost:3000](http://localhost:3000).
